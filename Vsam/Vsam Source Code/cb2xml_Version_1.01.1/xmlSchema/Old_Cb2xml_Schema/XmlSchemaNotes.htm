xmlSchema directory

This directory holds both dtd and xsd schema's for the Copybook-Xml produced by cb2xml. You can generate Jaxb classes from either with the XJC command (part of Java SDK, see generateJava.bat/rexx).

  • cb2xml.dtd - dtd Schema for the Xml generated by cb2xml.
  • cb2xml.xsd - xsd Schema for the Xml generated by cb2xml.
