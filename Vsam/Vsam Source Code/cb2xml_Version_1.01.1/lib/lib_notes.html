Lib directory

This directory holds to jar files you can find examples of ther use in the Examples Directory. The jars in this directory are:

  • cb2xml.jar - the main cb2xml jar.
  • cb2xml_definitions.jar - Holds constants used in cb2xml and dialect definitions.
    • It is useful if you wish to process a xml-file-description generated by cb2xml (without the full cb2xml jar).
  • ObscureCopybook.jar - Obscure a Cobol-Copybook, i.e Convert group / field names to meaningless names. It allows you send in copybooks without revealing any corperate details.
  • cb2xml_Jaxb.jar Java code load cb2xml - Xml via JAXB. This is an addition to cb2xml. It provides CobolParser and XmlParse to convert Copybooks in to Copybook/Item java classes. See the following for an example of using this jar.
