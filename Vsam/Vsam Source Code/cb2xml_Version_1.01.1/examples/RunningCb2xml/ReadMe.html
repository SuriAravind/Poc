RunCb2xml Directory

This directory holds example bat/sh files to run cb2xml.

  • cobol2xmlFileDescription.bat/sh - Converts a Cobol Copybook's to Xml
  • data2xml.bat/sh Convert a Cobol-Data file to Xml using a Cobol copybook. It only handles small text files. You are much better off using the Data2Xml program in the JRecord project
  • xml2data.bat/sh Convert xml file to a cobol data fille. It only handles text files, no duplicate fields, no arrays. You are much better off using the Xml2Data program in the JRecord project
  • ObscureCopybook.bat This will change all the field names to fairly meaningless names. This allows you to send in copybooks without revealing corporate details.
