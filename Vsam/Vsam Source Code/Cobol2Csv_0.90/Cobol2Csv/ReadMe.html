ReadMe - CobolToCsv

This project will convert single-record-type Cobol Data files to/from Csv Files. Also Multi-Record-Type can be converted to Csv.

Project documentation is available here.

The CobolToCsv project depends on:

  • cb2xml - Converts a Cobol Copybook to Xml.
  • JRecord - Read/Write Cobol data files using a Cobol Copybook in Java. It also supplies the Csv file handling.
