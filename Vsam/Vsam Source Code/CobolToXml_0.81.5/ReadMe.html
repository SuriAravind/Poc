ReadMe - CobolToXml

This project will convert single-record-type Cobol Data files to/from Xml Files.

Project documentation is available here.

The CobolToXml project depends on:

  • cb2xml - Converts a Cobol Copybook to Xml.
  • JRecord - Read/Write Cobol data files using a Cobol Copybook in Java.
