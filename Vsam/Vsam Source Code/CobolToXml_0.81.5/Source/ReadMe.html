Cobol To Xml Source

This project uses the source from 2 other projects

  • cb2xml - Cobol Copybook to Xml Analysis a cobol copybook and converts it into Xml
  • JRecord - Cobol File IO it lets you read/write a Cobol-data-File using a Cobol Copybook.

In version 0.81.4a the source for cb2xml and JRecord has been included (othersource directory) as there will be no JRecord release
